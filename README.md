# Customer Churn Prediction Using Machine Learning

## Project Overview
This project predicts customer churn using Machine Learning techniques on the IBM Telco Customer Churn dataset.

## Objectives
- Analyze customer data
- Clean and preprocess the dataset
- Build a Logistic Regression model
- Predict customer churn
- Evaluate model performance

## Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
IBM Telco Customer Churn Dataset

## Model Used
- Logistic Regression

## Accuracy
82.04%
